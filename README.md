# uber

### Installing dependencies

```
$ pip install requirements.txt
```

### Enpoints associated

1) /drivers/ - GET, POST, PUT, DELETE
2) /passengers/ - GET, POST, PUT, DELETE
3) /cabs/book/ - POST, PUT
4) /get-token/ - POST
5) /cabs/history/ - GET
